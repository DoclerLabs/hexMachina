What is hexMachina?

A powerful multi-modular MVC framework written in [Haxe](http://haxe.org "haxe.org").

What is it for?

For building configurable, large scaling modular applications, games, stateful websites and servers, targeting differnet platform. hexMachina is an OpenSource project, free to use and collaborate with it.

Try it!

Install haxelib install hexmachina
